12class NeuralNetwork(_NNCore, ClassifierMixin): 

13 """Class for defining neural network classifier weights optimization 

14 problem. 

15 

16 Parameters 

17 ---------- 

18 hidden_nodes: list of ints 

19 List giving the number of nodes in each hidden layer. 

20 

21 activation: str, default: 'relu' 

22 Activation function for each of the hidden layers. Must be one of: 

23 'identity', 'relu', 'sigmoid' or 'tanh'. 

24 

25 algorithm: str, default: 'random_hill_climb' 

26 Algorithm used to find optimal network weights. Must be one 

27 of:'random_hill_climb', 'simulated_annealing', 'genetic_alg' or 

28 'gradient_descent'. 

29 

30 max_iters: int, default: 100 

31 Maximum number of iterations used to fit the weights. 

32 

33 bias: bool, default: True 

34 Whether to include a bias term. 

35 

36 is_classifier: bool, default: True 

37 Whether the network is for classification or regression. Set 

38 :code:`True` for classification and :code:`False` for regression. 

39 

40 learning_rate: float, default: 0.1 

41 Learning rate for gradient descent or step size for randomized 

42 optimization algorithms. 

43 

44 early_stopping: bool, default: False 

45 Whether to terminate algorithm early if the loss is not improving. 

46 If :code:`True`, then stop after max_attempts iters with no 

47 improvement. 

48 

49 clip_max: float, default: 1e+10 

50 Used to limit weights to the range [-1*clip_max, clip_max]. 

51 

52 restarts: int, default: 0 

53 Number of random restarts. 

54 Only required if :code:`algorithm = 'random_hill_climb'`. 

55 

56 schedule: schedule object, default = mlrose_ky.GeomDecay() 

57 Schedule used to determine the value of the temperature parameter. 

58 Only required if :code:`algorithm = 'simulated_annealing'`. 

59 

60 pop_size: int, default: 200 

61 Size of population. Only required if :code:`algorithm = 'genetic_alg'`. 

62 

63 mutation_prob: float, default: 0.1 

64 Probability of a mutation at each element of the state vector during 

65 reproduction, expressed as a value between 0 and 1. Only required if 

66 :code:`algorithm = 'genetic_alg'`. 

67 

68 max_attempts: int, default: 10 

69 Maximum number of attempts to find a better state. Only required if 

70 :code:`early_stopping = True`. 

71 

72 random_state: int, default: None 

73 If random_state is a positive integer, random_state is the seed used 

74 by np.random.seed(); otherwise, the random seed is not set. 

75 

76 curve: bool, default: False 

77 If bool is True, fitness_curve containing the fitness at each training 

78 iteration is returned. 

79 

80 Attributes 

81 ---------- 

82 fitted_weights: np.ndarray 

83 Numpy array giving the fitted weights when :code:`fit` is performed. 

84 

85 loss: float 

86 Value of loss function for fitted weights when :code:`fit` is 

87 performed. 

88 

89 predicted_probs: np.ndarray 

90 Numpy array giving the predicted probabilities for each class when 

91 :code:`predict` is performed for multi-class classification data; or 

92 the predicted probability for class 1 when :code:`predict` is performed 

93 for binary classification data. 

94 

95 fitness_curve: np.ndarray 

96 Numpy array giving the fitness at each training iteration. 

97 """
